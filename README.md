# 逃出升天 (TaoChuShengTian)

> 基于 Unreal Engine 5.5 开发的第一人称生存恐怖游戏 Demo。
> 纯 C++ 实现核心玩法系统，采用工业级数据驱动与组件化设计。

## 项目简介

本项目是我个人独立开发的 UE5 C++ 学习项目，旨在打造一个完整的第一人称游戏核心框架。所有系统均遵循**数据与逻辑分离、组件化、低耦合**的软件工程原则。

## 技术栈

- **引擎版本**：Unreal Engine 5.5
- **编程语言**：C++ 20
- **核心技术**：
  - 增强输入系统 (Enhanced Input)
  - 多播委托与事件驱动架构
  - 数据资产 (Data Asset) 驱动的物品系统
  - 组件化设计模式
  - 智能指针与内存安全管理

## 已实现功能

### 角色控制系统
- 第一人称视角移动（前后左右、跳跃、蹲伏）
- 鼠标视角控制
- 按住 Shift 加速跑
- 弹簧臂摄像机挂载到角色头部骨骼插槽

### 物品拾取系统
- 基于球形碰撞体的范围检测
- 物品高光提示效果
- 多播委托解耦的交互机制
- 精准的委托绑定与解绑，彻底杜绝空指针崩溃
- 自动拾取最近的物品

### 武器装备系统
- 数据资产驱动的武器配置
- 拾取武器后自动装备到手部插槽
- 支持切换不同类型的武器
- 鼠标左键攻击输入框架

### 数据资产系统
- 统一的物品数据资产基类
- 支持不同类型物品的扩展
- 所有游戏数据与逻辑完全分离
- 策划可在编辑器中直接配置物品属性
