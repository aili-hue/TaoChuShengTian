逃出生天 (TaoChuShengTian)
项目简介
这是一个使用 Unreal Engine 5 C++ 开发的第三人称密室逃脱类游戏 Demo，是我个人学习 UE C++ 过程中的实战练习项目。
做这个项目主要是为了熟悉 UE 引擎的开发流程，练习组件化架构、委托解耦、碰撞交互、数据资产驱动等核心机制。目前项目还处于早期开发阶段，功能和代码都存在不少不足，后续会持续迭代优化。
已实现的基础功能
基于增强输入系统（Enhanced Input）的角色基础控制：移动、跳跃、蹲伏、奔跑
组件化设计的背包系统，支持物品拾取与数据管理
近战武器的挂载、切换与攻击碰撞检测逻辑
物理约束门交互，包含过门视角自适应、移动速度限制等体验优化
基于数据资产（Data Asset）的物品配置管理
角色血量组件与基础伤害判定
开发环境
引擎版本：Unreal Engine 5.5
开发方式：C++ 与蓝图混合开发
