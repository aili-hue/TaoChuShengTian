TaoChuShengTian（逃出生天）
一个基于 UE5.4 的第三人称逃生游戏练习项目，主要用来学习和实践 C++ 与蓝图混合开发。

该项目是我自学 UE 5 个月后的第一个完整项目，目前仍在持续迭代中。代码开源，欢迎指正。

项目简介
逃出生天是一个练习性质的单机游戏 Demo。玩家需要在场景中寻找武器、管理背包、与环境交互，对抗敌人并完成逃生目标。

这个项目不是为了“做一个完整的游戏”，而是为了在一个真实项目里把学到的知识都用一遍。目前所有核心逻辑都用 C++ 实现，部分 UI 和动画在蓝图中完成。

目前已实现的功能
角色控制：基于增强输入系统，实现移动、跳跃、加速、蹲下

一/三人称视角切换：支持平滑过渡，武器持有状态自动调整视角

背包系统：基于组件 + 数据资产，支持物品拾取、存储、切换

武器系统：近战武器生成、装配、攻击判定，动画蒙太奇切换

交互系统：门与环境物体的 Overlap 检测、委托通知、状态限制（进门时限制攻击/降低速度/切换视角）

血量系统：组件化管理，支持伤害计算与 UI 联动（预留接口）

交互反馈：简单的 UI 血条更新逻辑

技术栈
引擎：Unreal Engine 5.4

语言：C++（核心逻辑 90%）+ 蓝图（UI 与部分动画）

核心特性：增强输入、组件化架构、数据资产、委托与接口、定时器、弱指针与 GC 管理
